## **직사각형 별찍기** 

### **문제** 

이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다.  
별(\*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.


### **제한조건**

n과 m은 각각 1000 이하인 자연수입니다.


### **입력 / 출력**

|입력1  |입력2  |출력            |  
|:-----:|:-----:|:-----:        |
|5      |3      |\*\*\*\*\*  <br/> \*\*\*\*\* <br/>  \*\*\*\*\*  |



### **풀이**

```js
//별찍기
function star(a, b){
    let aStart ='';
    for(let i = 0; i< a; i++){
        aStart += '*'             
    }
    
    for(let j =0; j<b; j++){
        console.log(aStart);
    }
}
star(5,3);
```

```js
//다른사람 풀이
function star2(a, b){
    const row = '*'.repeat(a);

    for(let i =0; i<b; i++){
        console.log(row);
    }
}
star2(5, 3);
```

### **정리**
- repeat() : 메서드는 문자열을 주어진 횟수만큼 반복해 붙인 새로운 문자열을 반환 - str.repeat(count);
