### **스코프**
스코프란 유효범위/접근 가능한 범위로 

>#### **전역 스코프(Global scope)**
> 코드 어디서든 참조 가능


>#### **지역 스코프(Local scope)**
> 함수 코드 블록이 만들 스코프로 자신과 하위 함수에서만 참조 가능
> ex) 함수 코드 블록? if, for, while, try/catch 등
