### **Context**
리엑트에 내장된 내부적인 state 저장소로, props를 넘겨주지 않고, 컴포넌트 트리 전체에 데이터를 제공할 수 있음(전역적 데이터 공유)  
ex) 로그인, 테마, 웹 내부 설정파일 등등 사용







### **Context 동적생성**
```<AuthContext.Provider>```
