### **HTML 문서 구조** 

```html
<!DOCTYPE html>  <!-- html5 문서 -->
<html lang="ko"> <!-- 언어는 한국어 -->
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style></style> <!-- 스타일 영역 -->
    <script></script> <!-- 스크립트 영역 -->
</head>
<body> <!-- 내용 영역 -->
    
</body>
</html>
```
